About This Map

Regen Events Map 2026

A community-curated map of regenerative gatherings, ecovillage festivals, bioregional congresses, permaculture convergences, and adjacent events happening around the world in 2026.

This project emerged from the regenerative economy ecosystem — connecting communities working on bioregionalism, decentralized governance, community coordination, regenerative finance, and systems change.

✨ Resonance Beacons

When you star an event, it becomes a "resonance beacon" — a signal visible to the whole community showing interest in that gathering. You can see how many people are interested in each event and who they are. This helps the community self-organize and find each other on the road.

🔒 Privacy & Identity

We don't track you. There are no accounts, no cookies, no analytics. Your "ignored" events list is stored only in your browser's local storage — nobody else can see it, and it disappears if you clear your browser data.

If you star an event, you can choose to share your name or star anonymously. Names are stored in our database so others can see who's interested, but there's no email, no login, no personal data beyond the name you choose to give.

➕ Adding Events

Anyone can suggest an event by clicking "+ Suggest Event" and dropping a link. Suggestions go into a review queue — a human checks them and adds the full details to the map. You can also contribute via our Telegram group where a bot collects shared event links automatically.

🛠️ How It's Built

Open source. Single HTML file hosted on GitHub Pages. Event data stored in Supabase (open-source PostgreSQL). Map tiles by OpenStreetMap + CARTO. No frameworks, no build step, no vendor lock-in.

🐛 Found a Bug?

Click any event and use the "Report" button to flag wrong dates, broken links, or missing info. Reports go directly to our review queue.
